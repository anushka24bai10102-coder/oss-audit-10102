## Project Description
This project is based on understanding open-source software and practicing Linux shell scripting. It includes 5 scripts that perform different system tasks.

---

## Scripts

1. system_info.sh  
Displays system details like kernel, user, uptime, and OS.

2. package_check.sh  
Checks if Git is installed and shows version info.

3. disk_audit.sh  
Shows directory permissions and disk usage.

4. log_analyzer.sh  
Counts keyword occurrences in log files.

5. manifesto.sh  
Generates a personal open-source manifesto.
